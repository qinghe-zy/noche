# Tech 目录说明

用途：存放技术架构、数据模型、AI 协作规则与工程级约束。

主验收设备：
- `Redmi Note 11T Pro+`
- 当前 Android 实机体验、后台恢复、隐私锁恢复与静默暂存，优先围绕该机型验收。

命名规则：
- 架构使用 `architecture.md`
- 数据模型使用 `data_model.md`
- AI 协作使用 `ai_workflow.md`
- 其他技术说明以 `tech_*` 命名

当前补充：
- `noche_codex_function_matrix_and_interaction_logic.md`：Codex 执行用功能矩阵、主链优先级与交互硬规则
- `tech_frontend_backend_contract.md`：前端/后端分离开发契约
- `tech_backend_feature_checklist.md`：后端功能清单与推进顺序

使用方式：
- 先读 `architecture.md` 获取分层与协作边界
- 再读 `data_model.md` 了解核心对象和规则映射
- 再读 `noche_codex_function_matrix_and_interaction_logic.md` 锁定当前版本必须完成的主链与页面职责
- AI 协作请以 `ai_workflow.md` 为准
