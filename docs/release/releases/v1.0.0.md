# 寄屿 v1.0.0

`寄屿` 是一款 Android-only、local-first 的私人写信与日记应用。

## 这一版包含

- 日记、随笔、致未来 三条正文写作链路
- 本地图片插入、删除、编辑态展示、保存恢复、阅读态查看
- 本地草稿暂存与恢复
- 本地 SQLite 持久化
- 本地备份导出能力
- 信箱、日历、个人页、个人相册主路径
- 中英文语言切换
- 浅色 / 深色 / 跟随系统主题切换

## 本次正式版修复重点

- 修复深色模式切换后多页面仍停留在浅色固定值的问题
- 修复图片正式保存后被误删，导致阅读态和相册看不到的问题
- 修复 Diary 天气 / 心情前序图标在正式 APK 中丢失的问题
- 修复个人相册图片缩略图与大图查看链路
- 修复本地备份导出路径与 SQLite 重开问题
- 将首页右上角入口改为明确可点的 `个人主页`
- 优化 Diary / Jotting / Future 的键盘稳定性、纸面延展与文字贴线手感
