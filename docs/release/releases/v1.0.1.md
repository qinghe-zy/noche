# 寄屿 v1.0.1

`寄屿` 是一款 Android-only、local-first 的私人写信与日记应用。

## 这一版包含

- 日记、随笔、致未来 三条正文写作链路
- 本地图片插入、删除、编辑态展示、保存恢复、阅读态查看
- 本地草稿暂存与恢复
- 本地 SQLite 持久化
- 本地备份导出与导入能力
- 信箱、日历、当日归档、个人页主路径
- 中英文语言切换
- 深色模式基础支持

## 当前发布资产

- Android release APK
- 包名：`com.qinghezy.jiyu`
- 版本号：`1.0.1`
- versionCode：`101`

## 安装说明

1. 使用本页提供的 APK 安装到 Android 设备
2. 若设备里已有旧版本，建议先卸载旧包后再安装
3. 首次启动后确认应用内没有残留历史测试数据
