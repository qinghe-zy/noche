Chinese Font Strategy
Chinese typography should carry the same editorial voice.

If we only tune color and spacing, the app still loses Claude's personality in Chinese. Typography has to be designed as part of the theme.

Chinese Font Pairing

Heading / Editorial Serif
在纸页之间，今天的心绪也值得被认真排版。

UI / Body Sans
外观设置、日历、信箱、编辑器正文说明、按钮、标签、状态信息统一走中文无衬线，保证可读性和整站秩序。
Recommended stack: `Source Han Sans SC`, `Noto Sans SC`, `PingFang SC`, `Microsoft YaHei`, `sans-serif`
