What changes everywhere

Design constants

  • Serif headlines, sans UI text
  • Warm neutral palette, no cool blue-gray drift
  • Ring shadows instead of heavy drop shadows
  • More generous section spacing and softer radii

Implementation implications

  • Move pages away from hardcoded color values
  • Promote shared surface, text, border, and accent tokens
  • Split settings UI into theme family and mode
  • Refit shared dialogs and topbars before polishing pages
