# 寄屿

`寄屿` 是一款写给自己的 Android 记录应用。

它把日记、随笔和写给未来的信放进同一套安静的本地写作体验里：不依赖服务器，不要求登录，默认离线可用，重点保证“能写、能存、能恢复、能回看”。

## 这款应用能做什么

- 写日记
- 记随笔
- 给未来的自己留一封信
- 插入本地图片并随正文一起保存
- 在本地恢复草稿和正式内容
- 浏览信箱、日历、当日归档和个人相册
- 导出和导入本地备份

## 当前产品特点

- Android-only
- local-first
- 离线可用
- 不接服务器
- 不要求账号登录
- 数据保存在本地 SQLite 与本地文件中

## 下载与安装

当前版本：`v1.0.0`

发布包为 Android release APK。建议从 GitHub Releases 页面下载对应版本的 APK。

安装前建议：

1. 如果设备里已有旧版本，先卸载旧包
2. 再安装本次新的 release APK
3. 首次启动后确认应用内没有历史测试数据残留

## 隐私与数据

- 当前阶段所有核心能力都以本地为主
- 不依赖远程 API
- 不做云同步
- 备份与恢复也走本地文件链路

## 当前版本说明

当前主验收设备是 `Redmi Note 11T Pro+`。这一阶段优先保证：

- 可安装
- 可启动
- 可编辑
- 可保存
- 可恢复
- 可浏览
